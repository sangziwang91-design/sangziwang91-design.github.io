Research Focus

I study interaction-induced AI behavior under restricted-access conditions: settings where evaluators cannot inspect model weights, hidden states, logits, or proprietary telemetry, but still need to assess whether model behavior is stable, honest, and epistemically bounded.

My public work focuses on epistemic boundary assessment, output-layer behavioral deformation, and evaluation protocols as behavioral environments. Core public frameworks include EC-EpC, Adaptive Behavioral Scoring (ABS), BRC-8, Testing-as-Environment, Interaction Ecology, Dynamic Rule Spectrum, and Convergence Inertia.

The private engineering line extends these methods into local runtime prototypes for session monitoring, interaction-state tracking, behavioral routing, validation governance, and lightweight interaction-field monitoring and governance experiments. Public releases disclose methodology and claim boundaries; operational triggers, thresholds, routing logic, seed mechanisms, and intervention heuristics remain private.

Current status: Public methodology records (7 Zenodo DOIs) + local runtime prototype (GBSF Console v1.0, IDFS) + sandbox-stable validation. External validation and reproducible benchmark packaging remain the next step.

Research Stack

Public Methodology

EC-EpC
Epistemic Capability / Epistemic Claim Gap
Dual-axis framework for measuring divergence between what a model behaviorally demonstrates and what it claims or implies about its own knowledge boundary.
Public
ABS
Adaptive Behavioral Scoring
Output-layer behavioral deformation measurement framework using Drift, Resample, and StructureBreak components under restricted-access conditions.
Public
BRC-8
Behavioral Response Classification
Eight-class taxonomy for classifying observable response patterns: compliant convergence, boundary escape, principled refusal, meta-execution, confidence fabrication, selective omission, structural deference, and epistemic anchoring.
Public
Testing-as-Environment
Evaluation Protocols as Behavioral Environments
Methodological principle that evaluation sequences are not neutral probes of static capability, but environments that can elicit, shape, and reveal model behavior.
Public
IEF
Interaction Ecology Framework
Framework for analyzing long-context human–model interaction as an ecology of users, prompts, memory, protocols, model behavior, and feedback loops.
Public
DDRS
Dynamic Rule Spectrum System
Methodological line for analyzing dynamic rule pressure, interaction-induced constraint effects, and rule-spectrum behavior in LLM outputs.
Public
Convergence Inertia
Cross-Model Structural Drift
Pilot evidence line studying how models may converge toward stable response structures under repeated or structured interaction pressure.
Public

Generative Path Layer

GEO
Generative Engine Optimization / Path-Attraction Theory
Conceptual research line for studying how structured interaction conditions create path attraction, recurrence, closure, escape, and oscillation zones in model generation trajectories. Operational seed mechanisms remain private.
Conceptual Core
GBSF
Generative Behavior Science Framework
Operational measurement layer for GEO path-attraction dynamics. Converts interaction traces into measurable behavioral shifts: recurrence, variation, closure, escape, and tension retention.
Measurement Layer

GBSF operationalizes GEO concepts into deployment-accessible behavioral measurements under restricted-access conditions.

Human–Model Co-Interaction Layer

HMCGF
Human–Model Co-Generative Field
Conceptual field framework for understanding human–model interaction as a co-generative process, where user intention, model response, memory, protocol, and evaluation pressure jointly shape interaction trajectory.
Field Framework
HM-CST
Human–Model Co-Sculpting Topology
Topology-inspired framework for studying repeated human–model interaction dynamics: post-interaction residue, interaction lock-in, observer-induced shifts, and cross-scale diffusion. Proposes computational proxies; not a completed mathematical theory.
Interaction Dynamics

HM-CST is not presented as a proven mathematical theory or a direct measurement of human cognition. It is a topology-inspired framework with proposed computational proxies such as zigzag persistence, sheaf-laplacian consistency, interaction-residue indices, and observer-induced shift measures.

Scope — What This Is / Is Not

This work is

  • Restricted-access behavioral observability
  • Output-layer and interaction-layer methodology
  • Public conceptual records with local runtime prototypes
  • Sandbox-stable engineering validation
  • A public/private research architecture with clear disclosure boundaries

This work is not

  • Direct model-internal interpretability
  • A claim of solved AI alignment
  • A production deployment platform
  • A validated civilization-scale governance system
  • A public release of the private runtime core

Claim Ceiling

The current public claim is restricted-access behavioral observability and sandbox-level runtime validation. This work does not claim direct access to model internals, causal control over model distributions, solved alignment, or validated civilization-scale governance.

Public records establish methodology, terminology, pilot evidence, and engineering viability. Real-model causal intervention effects, cross-model reproducibility, and population-level validity require further validation.
