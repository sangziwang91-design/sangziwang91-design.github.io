Research Focus

I study interaction-induced AI behavior under restricted-access conditions: settings where evaluators cannot inspect model weights, hidden states, logits, or proprietary telemetry, but still need to assess whether model behavior is stable, honest, and epistemically bounded.

My public work focuses on epistemic boundary assessment, output-layer behavioral deformation, behavioral scoring datasets, and evaluation protocols as behavioral environments. Core public frameworks include EC-EpC, Adaptive Behavioral Scoring (ABS), BRC-8, Testing-as-Environment, Interaction Ecology, Dynamic Rule Spectrum, Convergence Inertia, and GEO-LINE-X.

The private engineering line extends these methods into local runtime prototypes for session monitoring, interaction-state tracking, behavioral routing, validation governance, and lightweight interaction-field monitoring and governance experiments. Public releases disclose methodology and claim boundaries; private runtime mechanisms, internal routing logic, scoring thresholds, intervention heuristics, raw logs, and unpublished validation workflows are intentionally withheld.

Current status: Public methodology and dataset records (10 Zenodo DOI deposits) + local runtime prototypes + external-bank BRC pilot evidence. A 1,000-record external-bank run has completed with zero run errors, all configured source groups covered, and BRC-1 through BRC-8 observed; follow-up calibration/retest work is in progress. Current public claim remains pilot evidence and calibration substrate, not peer-reviewed validation or causal proof.
